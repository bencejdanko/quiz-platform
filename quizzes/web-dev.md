What does HTML stand for?
- [ ] Hyper Text Markup Language
- [x] HyperText Markup Language
- [ ] High Tech Modern Language

> HTML (HyperText Markup Language) is the standard markup language for creating web pages. 
> The correct spelling has no space between "Hyper" and "Text".

Which languages run in the browser?
- [x] JavaScript
- [ ] Python
- [x] WebAssembly
- [ ] C++

> JavaScript is the primary scripting language for web browsers. WebAssembly (Wasm) is a newer binary instruction format that also runs in modern browsers, enabling high-performance applications. While Python and C++ can be compiled to WebAssembly, they don't run natively in browsers.
